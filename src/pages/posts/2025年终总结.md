## 2025 年终总结

> 我的2025，略有遗憾，不失圆满

### 一、考研后记

在2024年十月份放弃考研这件事开始，似乎也并没有很失落，为了考研学习的英语增加的六千个词汇量，以及计算机数据结构，对我的一生都是有益的。

既然梦校无望、本校无意，就开始考虑工作的事情吧。

在年初，跟随学习[短链接项目](https://github.com/black8848/SaaS_ShortLink),每天和好朋友一起去学校的图书馆学习，进度很慢，最终也没有完成这个项目。

一直持续到三月份假期回家休息，投投简历试试吧，楠楠说。

### 二、工作

就投了一份简历，但是第二天就通知面试。

好吧，去面吧。

考官，也就是后来我的直系领导，技术面，考察了我的技术栈，具体问了什么已经回忆不起来了，只记得面完觉得自己有很多问题都答不上来，出公司后打个车回去了。

半个月之后，通知我初步面试通过了，心里还是挺高兴的，虽然实习工资开的很低，但是去试试总不是坏事。

三月二十四日，我的工作阶段开始了。

很幸运，我有一个很好的直属领导，我叫他张哥，张哥对于技术的发展很敏锐，对于新技术有很强的兴趣，很欣赏「爱折腾」的特质，由此，我开始了研发工作。

三月份到六月份之间这三个月，几乎是自由学习时间，我自由的在了解技术栈，偶尔张哥会问问目前学到哪里了，除此之外几乎是自由学习，在这期间，我上手了 C# 语言，并在今年 .NET 几乎成了我的主要技术栈。

之后参与了公司的项目的后端开发，参与开发了两个项目。

参与开发时，我很享受这种在发挥自己的时候，我很喜欢那种氛围，感受到我被需要，我在发热。

自己和同事配合 Vibe Coding 全新开发了一款产品，并选购硬件，实现了软硬件结合，也许在明年经过测试后，这款产品会成为我和同事全新制作的公司产品，很期待那一刻。

### 三、生产资料

因为公司的性质，所以在配电脑时，只能在Windows中选择，最后选择了 ThinkPad T14P GEN 3 U9-285H 64G 1T 版本，这台电脑的内存对我来说很够用，处理器的性能也可以，除了做工很有一股塑料感。

在开发上，由于我已经习惯了 Mac，而且注意到微软在 2023 年 9 月对 WSL 新增了「镜像网络」的支持，使得 WSL 子系统可以与WIndows 主系统共用同一网卡，也就是我在 WSL 启动我的后端服务，别人只需要连接我的 Windows 的 IP 即可正常访问了。

WSL2 的这项功能的增加，解决了我对于 WSL2 的网络部分的忧虑，所以我在 Windows 使用 WSL2 开发。

Shell 使用了我在Mac上熟悉的 zsh + oh-my-zsh 组合，搭配好如 p10k、zsh-autosuggestions、zsh-history-substring-search、zsh-syntax-highlighting 等插件，很好用。

IDE 在尝试了 Rider，不是很喜欢微软的 Visual Studio，UI 实在是有些凌乱，如同 Windows 11系统一样，有很多繁杂的功能，但是因为 WSL2 网络镜像的功能仅在 Win 11 上支持，所以我到手对 Win 11 修改了很多

因为我习惯了在终端启动程序，所以我是直接在 WSL 子系统中构建启动，对程序非常友好，但是通过 Rider 启动，由于我的 Rider 在 Windows 系统下，所以构建生成的是 Windows 下的obj/、bin/，我也知道  Jetbrain 对 WSL 支持也很好，通过 Jetbrain Gateway 转由 WSL 加载项目，遇到了一些其他的问题，但是我习惯用终端启动项目，Rider只是用来修改代码。

但是 IDE 可能是一直在加载索引？CPU一直顶满我的CPU占用，导致我的电脑很卡，我没有尝试太多次，我直接卸载了 Rider，兜兜转转，还是回到了 VS Code + 各种插件。

字体选择上，英文选择Cascadia Code等宽字体，中文发现 HarmonyOS Sans 的中文字体意外的不错，于是最终选择了 Cascadia Code, HarmonyOS Sans Light SC。

### 四、毕业

三月份，开始钻研毕业设计和毕业论文，选择了机器学习相关的课题「人群计数系统的设计与实现」，使用了 YOLO 技术栈，经过一百多轮的训练，实现了一个简单有效的人体识别系统。

我很喜欢我的指导老师，我的指导老师很细心，对论文的格式一丝不苟，很感谢她在几个月的时间内对我的论文格式的培养，这对我以后的工作或者文章报告的格式都很重要，很感谢她。

四月份之后，一边工作一边攥写毕业论文，在六月八号，最终的毕业论文答辩中，我对我的项目自信的娓娓道来，流畅的回答了评审老师的问题，至此，顺利毕业了。

### 五、生活

#### 1. 理想

曾经和舍友们在一起玩，大家在晚上散步的时候，大家说起自己的未来的生活，有要去考研到大城市闯一闯的，有要留在当前城市发展的，也有很强但是在哪随意的。

我当时说道，我不期待未来工资很高，只希望能有自己的时间，过上早八晚五正常休息、和爱人一起慢慢的生活。

现在实现了，我经常会做一些喜欢做的、喜欢吃的菜，也会简单炒点小菜饱腹，喜欢的衣服可以随意拿下，想要的想玩的想吃的，基本都可以拿下，活在当下，也许这就是我最喜欢的生活了。

#### 2. 学习

工作稳定了，生活稳定了，反而更喜欢学习了，开始很感兴趣计算机网络、 Linux 内核、密码学。

##### 2.1 计算机网络

截止目前12月21日，看完了五分之一的 「图解 TCP/IP」。

##### 2.2 计算机内核

还没具体看 Linux 内核，但是跟随一本小书 [1000 行代码的操作系统](https://1000os.seiya.me/zh/) 开始学习实现一个小型的 [RISC-V 系统内核](https://github.com/black8848/OSin1000Lines)，目前做完了「启动内核、实现了C标准库，可以进行输出语句、PANIC宏、异常、内存分配器、进程控制」了解了很多计算机内核上的知识。

##### 2.3 密码学

密码学知识，起初是在 V2EX 上看到一个人的帖子的争论，然后引到评论区大家的评论，了解到一个大佬的博客:[This Cute World](https://thiscute.world/),在他的博客中，我学习了他的[系列 - 写给开发人员的实用密码学](https://thiscute.world/series/%E5%86%99%E7%BB%99%E5%BC%80%E5%8F%91%E4%BA%BA%E5%91%98%E7%9A%84%E5%AE%9E%E7%94%A8%E5%AF%86%E7%A0%81%E5%AD%A6/),写的很好，很推荐。

在密码学中学会了很多，如不对称加密如 RSA、ED25519、对称加密 AES、加盐方式、迭代算法 KDF、HASH等。

然后进而了解到：

- 服务器是如何存储用户的密码以及用户登陆是如何校验的

- 为什么 ssh 推荐密钥登录而不是密码登录
- 加密货币是基于什么原理
- 数学难题是如何应用于现实的，曾经我只觉得数学难题是不好的，但是对于密码学来说，加密原理就是基于目前看来不可能被破解的数学难题为基础来建立的，所以数学难题还是有益处的，如果没有数学难题，那么密码如何存储，就又是个令人头疼的问题了。

我的学习方式就是不断的问 chatGPT，对于不懂的知识，一边搜索，一边不断的问问题，每天的生活中，我都称呼 chatGPT为 「G老师」。

##### 2.5 英语

对于英语，一直很感兴趣，但是学习英语对于我来说还没有明确的路线，目前在看[离谱的英语学习指南](https://github.com/byoungd/English-level-up-tips)，写的还不错，但是还没有看完。

##### 2.6 一个小项目 - noemoji

Vibe Coding 经常生成大量的 emoji，做了一个小工具，可以快速扫描并去除项目中的 emoji，我给项目取名为 [noemoji](https://github.com/black8848/noemoji)。

关于项目介绍，发在了 V2EX 帖子中：[写了一个小工具，用于去掉项目中的 emoji](https://v2ex.com/t/1176448?p=1#reply3)。

只是一个根据自己需求开发的小工具，发布到了 Github 上，发布帖子时诚惶诚恐，没想到在 Github 上也收获了 10 个 Star（当然包括我自己一个 :)) )，很高兴能帮助到别人。

##### 2.7 科学

今年购买了两台服务器，分别是：

- 搬瓦工的 MiniChicken，无线路优化，17.71 刀/年，当玩具鸡/备用机用。
- DMIT 的 LAX.AN4.Pro.MALIBU，IPv4 三网 CN2GIA 线路优化，IPv6 也是三网优化，质量很高，解锁也还不错，价格是 49.9 刀/年

最近在观看[不良林](https://www.youtube.com/@bulianglin)的软路由系列，讲的很棒，仍在学习中。

#### 3. 交际

今年毕业开始，和楠楠一起生活了，我们的生活很好，一起吃饭、一起做家务、一起看剧，我很喜欢和楠楠在一起。我能感受到我在慢慢改变，也能感受到楠楠也在慢慢改变，我们的心更近了。

我和楠楠今年国庆互相见了家长，和楠楠的妈妈、姥姥一起包了饺子，在楠楠家很开心，只是很遗憾因为工作原因没有见到楠楠的爸爸。

今年和 Lihe 在秦皇岛租车玩了一个周末，玩的很开心，在海边开车的感觉很惬意，很棒的旅游体验。

今年认识了很多新朋友，认识了一个性格很棒的朋友，和他共同开发工作中的新产品。

和老朋友们进入了新一年。
